Summary: Latitude vs. Weather Parameters

The purpose of this project was to analyze how weather changes as you get closer to the equator.To accomplish this analysis we first pulled data from the Open Weather Map API, to assemble data on over 500 cities.




After asssembling the dataset, we used matplotlib to plot various aspects of the Weather vs Latitude. Factors we looked into included temperature, cloudiness, wind Speed and humidity.This site provides source data and visualization created as part of the analysis as well as explainations and descriptions of any trends and correaltions witnessed
